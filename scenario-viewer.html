Scenario Viewer

Click Load Scenario to open a scenario markdown file, or drag and drop a .md file onto this page.

The viewer renders the markdown with campaign-appropriate styling. Use Print / PDF to export a clean document for Google Drive.

Features

  • Tables are automatically styled — stat blocks detected and centred
  • Italic opening paragraphs render as centred flavour text
  • Blockquotes render as parchment callouts
  • Print mode switches to a clean light theme
